# Getting Started with Auditomatic Lite

## Overview

Auditomatic Lite is a browser-based tool for systematic LLM evaluation. Follow this guide to get up and running quickly.

## Basic Workflow

1. **Create a Design** - Define your prompt template with variables
2. **Add Variable Lists** - Create sets of test inputs
3. **Run Experiment** - Execute tests across multiple models
4. **Analyze Results** - Use pivot tables and export data

## First Steps

1. Open Auditomatic Lite in your browser
2. Click "Create New Design"
3. Write a prompt with variables like `{name}` or `{topic}`
4. Add a variable list with test values
5. Configure your API settings
6. Run your first experiment
